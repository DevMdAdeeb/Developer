# بوت إشارات العملات الرقمية

هذا المشروع هو بوت تلغرام يقوم بإرسال إشارات فنية للعملات الرقمية.

## كيفية التشغيل على Termux

لتشغيل هذا البوت على جهازك باستخدام Termux، اتبع الخطوات التالية:

### 1. تثبيت Termux
- قم بتحميل وتثبيت تطبيق [Termux](https://termux.com/) من متجر F-Droid.

### 2. تثبيت Python
- افتح Termux وشغّل الأمر التالي لتثبيت Python:
  ```bash
  pkg install python
  ```

### 3. تنزيل ملفات المشروع
- قم بتنزيل ملفات المشروع (أو استنسخها باستخدام `git` إذا كانت متوفرة على GitHub).
  ```bash
  # إذا كان لديك git مثبتًا
  # git clone <رابط المشروع>
  # cd <اسم المجلد>
  ```

### 4. وضع توكن البوت
- افتح ملف `token.txt` الذي قمت بتنزيله.
- احذف النص الموجود (`YOUR_TELEGRAM_TOKEN`) والصق توكن بوت التلغرام الخاص بك في الملف.
- احفظ الملف.

### 5. تثبيت المكتبات المطلوبة
- تأكد من وجودك في نفس المجلد الذي يحتوي على ملف `requirements.txt`، ثم شغّل الأمر التالي:
  ```bash
  pip install -r requirements.txt
  ```
  قد يستغرق هذا الأمر بعض الوقت لتثبيت جميع المكتبات.

### 6. تشغيل البوت
- أخيرًا، لتشغيل البوت، شغّل الأمر التالي:
  ```bash
  python main.py
  ```

### 7. إبقاء البوت يعمل في الخلفية
- إذا أغلقت Termux، سيتوقف البوت عن العمل. لإبقائه يعمل في الخلفية، يمكنك استخدام أداة مثل `tmux`.
- **لتثبيت tmux:**
  ```bash
  pkg install tmux
  ```
- **لبدء جلسة جديدة:**
  ```bash
  tmux new -s bot
  ```
- **لتشغيل البوت داخل الجلسة:**
  ```bash
  python main.py
  ```
- الآن يمكنك إغلاق Termux وسيبقى البوت يعمل. للعودة إلى الجلسة لاحقًا، افتح Termux وشغّل:
  ```bash
  tmux attach -t bot
  ```

بهذه الخطوات، يجب أن يعمل البوت بنجاح على جهازك.
