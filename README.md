# بوت إشارات العملات الرقمية

هذا المشروع هو بوت تلغرام يقوم بإرسال إشارات فنية للعملات الرقمية.

## كيفية التشغيل على Termux (تعليمات مبسطة)

بعد التحديثات الأخيرة، أصبحت عملية التثبيت أسهل بكثير. اتبع الخطوات التالية:

### 1. تثبيت Termux
- قم بتحميل وتثبيت تطبيق [Termux](https://termux.com/) من متجر F-Droid.

### 2. تحديث الحزم وتثبيت Python
- افتح Termux وشغّل الأوامر التالية:
  ```bash
  pkg update -y && pkg upgrade -y
  pkg install python -y
  ```

### 3. تنزيل ملفات المشروع
- قم بتنزيل ملفات المشروع (أو استنسخها باستخدام `git` إذا كانت متوفرة على GitHub).
  ```bash
  # إذا كان لديك git مثبتًا
  # git clone <رابط المشروع>
  # cd <اسم المجلد>
  ```

### 4. وضع توكن البوت
- افتح ملف `token.txt`.
- احذف النص الموجود (`YOUR_TELEGRAM_TOKEN`) والصق توكن بوت التلغرام الخاص بك في الملف.
- احفظ الملف.

### 5. تثبيت المكتبات المطلوبة
- الآن، عملية التثبيت أصبحت مباشرة ولا تتطلب بناء حزم معقدة. شغّل الأمر التالي:
  ```bash
  pip install -r requirements.txt
  ```

### 6. تشغيل البوت
- أخيرًا، لتشغيل البوت، شغّل الأمر التالي:
  ```bash
  python main.py
  ```

### 7. إبقاء البوت يعمل في الخلفية (اختياري)
- لإبقاء البوت يعمل في الخلفية حتى بعد إغلاق Termux، يمكنك استخدام `tmux`.
- **لتثبيت tmux:**
  ```bash
  pkg install tmux
  ```
- **لبدء جلسة جديدة:**
  ```bash
  tmux new -s bot
  ```
- **لتشغيل البوت داخل الجلسة:**
  ```bash
  python main.py
  ```
- الآن يمكنك إغلاق Termux. للعودة إلى الجلسة لاحقًا، افتح Termux وشغّل:
  ```bash
  tmux attach -t bot
  ```
