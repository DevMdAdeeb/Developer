# بوت إشارات العملات الرقمية

هذا المشروع هو بوت تلغرام يقوم بإرسال إشارات فنية للعملات الرقمية.

## كيفية التشغيل على Termux (التعليمات المحدثة)

لتشغيل هذا البوت على جهازك باستخدام Termux، اتبع الخطوات التالية بالترتيب:

### 1. تثبيت Termux
- قم بتحميل وتثبيت تطبيق [Termux](https://termux.com/) من متجر F-Droid.

### 2. تحديث الحزم وتثبيت الأدوات الأساسية
- افتح Termux وشغّل الأوامر التالية لتحديث الحزم وتثبيت الأدوات اللازمة لبناء المكتبات:
  ```bash
  pkg update -y && pkg upgrade -y
  pkg install python clang libffi-dev rust -y
  ```

### 3. تثبيت مكتبة Numba مسبقًا
- لتجنب أخطاء البناء التي واجهتها، يجب تثبيت مكتبة `numba` باستخدام `pkg` قبل استخدام `pip`. هذا الأمر سيقوم بتثبيت نسخة متوافقة مع Termux:
  ```bash
  pkg install python-numba -y
  ```

### 4. تنزيل ملفات المشروع
- قم بتنزيل ملفات المشروع (أو استنسخها باستخدام `git` إذا كانت متوفرة على GitHub).
  ```bash
  # إذا كان لديك git مثبتًا
  # git clone <رابط المشروع>
  # cd <اسم المجلد>
  ```

### 5. وضع توكن البوت
- افتح ملف `token.txt`.
- احذف النص الموجود (`YOUR_TELEGRAM_TOKEN`) والصق توكن بوت التلغرام الخاص بك في الملف.
- احفظ الملف.

### 6. تثبيت باقي المكتبات
- الآن، بعد تثبيت المتطلبات الأساسية، يمكنك تثبيت باقي المكتبات باستخدام `pip`:
  ```bash
  pip install -r requirements.txt
  ```

### 7. تشغيل البوت
- أخيرًا، لتشغيل البوت، شغّل الأمر التالي:
  ```bash
  python main.py
  ```

### 8. إبقاء البوت يعمل في الخلفية
- لإبقاء البوت يعمل في الخلفية حتى بعد إغلاق Termux، يمكنك استخدام `tmux`.
- **لتثبيت tmux:**
  ```bash
  pkg install tmux
  ```
- **لبدء جلسة جديدة:**
  ```bash
  tmux new -s bot
  ```
- **لتشغيل البوت داخل الجلسة:**
  ```bash
  python main.py
  ```
- الآن يمكنك إغلاق Termux. للعودة إلى الجلسة لاحقًا، افتح Termux وشغّل:
  ```bash
  tmux attach -t bot
  ```
